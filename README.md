# Financial Fraud Risk Analytics System

An end-to-end data analytics project analyzing credit card transaction data to detect fraud patterns using SQL and Python.

## Project Overview

This project analyzes **280,000+ financial transactions** to understand fraud behavior and identify risk patterns in transaction data.

The system combines **SQL-based analysis** with **Python exploratory data analysis** to uncover insights into fraudulent transactions.

## Technologies Used

- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Data Visualization
- Jupyter / Google Colab

## Dataset

Credit Card Fraud Detection dataset containing anonymized transaction features.

Total transactions analyzed: **284,807**

## Key Analyses Performed

### Fraud Distribution Analysis
Identified the proportion of fraudulent vs normal transactions.

### Transaction Size Analysis
Segmented transactions into small, medium, and large categories to observe fraud behavior.

### Fraud Probability Calculation
Computed fraud rates across transaction value groups.

### Time-Based Fraud Patterns
Analyzed transaction activity across time intervals.

### Correlation Analysis
Explored relationships between transaction features and fraud labels.

## Key Insights

- Fraud transactions represent **less than 0.2%** of all transactions.
- Small-value transactions show slightly higher fraud percentages.
- Certain anonymized variables show stronger correlations with fraudulent activity.
